Install

  • vagrant
  • virtualbox

Steps

  • Clone this repo. Goto vagrant subdirectory.
    • git clone https://github.com/kompseye/vagrant-ubuntu-nodejs.git
    • cd vagrant-ubuntu-tc
    • cd vagrant
    • vagrant up
  • SSH to locally running ubuntu! vagrant ssh
